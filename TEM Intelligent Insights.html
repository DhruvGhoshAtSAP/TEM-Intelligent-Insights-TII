Expected Data Format
10-column SAP SAC DMTP export — both full and scoped territory files are supported
#ColumnTypeDescription
1Idea Region L1DimensionTop-level geography: APAC, EMEA, Americas, MEE
2Idea Region L2DimensionSub-region: ANZ, G.China, India, BeNeLux, France, etc.
3Acc Owner NameDimensionTerritory Manager name (LASTNAME, Firstname)
4Idea NameDimensionSales Idea / campaign motion name
5Idea PartnerDimensionSAP partner — SI, reseller, or ISV
6Idea Sales Play NameDimensionSpecific SAP Sales Play driving the Idea
7Acc MasterCodeDimensionIndustry classification (SAP Master Industry Code taxonomy)
8Idea Solution Area L2DimensionSAP Solution Area: Finance, HCM, SCM, BTP, Data & AI, Cloud ERP, CX…
9# of Accounts per IdeaMeasureAccount count for this Idea/dimension combination
10Exp. PipelineMeasureExpected pipeline — primary financial KPI. Typically stored as EUR thousands (K) in SAP SAC exports. Use the unit selector above to set the correct scale.
The tool auto-detects your header row. Row 1 may contain measure group headers; Row 2 contains column names; Row 3+ is data. Both the full 97k-row global file and personal territory exports are supported.

A note from the team: EKX is amazing, but it has its limitations. TII required a whole lot of manual coding and bug fixing to get to where it is today. Feel free to use EKX for anything straightforward and easy to describe - but if you have something more complex in mind, please reach out to us directly using the email links in the Information & Credits tab. We are always happy to help!
